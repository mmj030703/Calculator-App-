Calculator App is made using HTML, CSS & JavaScript.
- It can perform operations like addition, subtraction, division, multiplication & finding mod Value.
- The user can't add a operator, a decimal point, single or double zero at start when the display is empty.
- The user can't add add more that one decimal point.
- The user can't put two operator together if try to do then the current operator will replace the previous one.
- The user can't divide by zero.
